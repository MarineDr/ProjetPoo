- Téléchargez sur Updago l’archive nommé DROIT_HOLTERMANN_VIVIER.zip. 
- Effectuez un clic droit sur le dossier puis extraire à l’endroit ou vous souhaitez que le jeu soit installé.
- Ouvrez un terminal et déplacer vous jusqu’au source du projet : cd …/realite_virtuelle/src
- Puis tapez la commande : java rv/Main  pour lancer le jeu

Si le jeu ne se lance pas, recompilez en lançant la commande : javac rv/*.java

Une fois le jeu lancé vous pouvez taper dans le terminal la commande aide pour obtenir un bref résumé des commandes.
Pour plus d'information veuillez consulter la documentation utilisateur.
